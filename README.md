# Student Course Management System

Backend project made using Node.js, Express.js and MySQL.

Features:
- Add Student
- Get Students
- Update Student
- Delete Student
- Get students by course

Tech Used:
- Node.js
- Express.js
- MySQL
- Thunder Client

API Routes:

GET /api/students

POST /api/students/add

PUT /api/students/update/:id

DELETE /api/students/delete/:id

GET /api/students/course/:id

Run project:

npm install

npm run dev
